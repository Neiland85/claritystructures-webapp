# Modelos de monetización posibles

## 1. Servicio premium manual-asistido

Venta inicial más realista.

Clarity se usa como soporte técnico y metodológico para ordenar material digital sensible.

Cliente tipo:

- despacho;
- empresa familiar;
- conflicto societario;
- caso con documentación digital compleja.

Ventaja:

- alto valor por operación;
- no exige producto 100% autoservicio;
- permite aprender del cliente real.

## 2. Piloto para despachos

Oferta cerrada:

- demo;
- onboarding;
- simulación con material no sensible;
- revisión de flujo;
- informe técnico de resultados.

Ventaja:

- bajo riesgo;
- feedback cualificado;
- abre puerta a contrato mayor.

## 3. SaaS vertical

Más adelante.

- onboarding;
- pricing recurrente.

## 4. Licencia enterprise

Para organizaciones que quieran instalar/controlar internamente.

Cliente tipo:

- corporate legal;
- compliance;
- ciberseguridad;
- aseguradoras;
- organismos.

## 5. Venta de activo / adquisición

El comprador adquiere:

- código;
- documentación;
- narrativa;
- pruebas;
- demo;
- potencial de mercado;
- know-how.

Sube de valor si hay:

- pilotos;
- métricas;
- clientes;
- validación externa;
- tráfico cualificado;
- pipeline comercial.
