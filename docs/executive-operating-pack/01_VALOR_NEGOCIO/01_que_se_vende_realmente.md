# Qué se vende realmente

## No se vende “una web”

La web es la puerta de entrada.

Lo que se vende es un sistema de confianza operativa para datos/evidencias digitales.

## Producto real

Clarity permite convertir material digital desordenado en un flujo más seguro:

1. entrada controlada;
2. clasificación inicial;
3. preservación contextual;
4. trazabilidad técnica;
5. generación de evidencia verificable;
6. documentación para revisión profesional;
7. límites claros de responsabilidad.

## Qué compra un cliente

Un cliente no compra “software”. Compra reducción de incertidumbre.

Compra:

- orden;
- criterio;
- protocolo;
- documentación;
- control;
- menor riesgo de manipulación;
- capacidad de explicar lo ocurrido;
- preparación para revisión técnica, legal o pericial.

## Qué compra un adquirente

Una empresa que quisiera comprar Clarity no estaría comprando solo código.

Estaría comprando:

- un activo técnico ya orientado a sectores sensibles;
- una narrativa de mercado;
- una arquitectura con trazabilidad;
- una base de documentación avanzada;
- un posible producto vertical para legaltech, govtech, ciberseguridad o auditoría;
- una entrada a servicios premium de alto margen.

## Dónde hay dinero

Hay dinero donde el error es caro:

- despachos con casos complejos;
- conflictos societarios;
- herencias con material digital;
- auditorías internas;
- brechas de datos;
- cumplimiento;
- departamentos legales;
- ciberseguridad;
- gobierno de evidencias;
- empresas que necesitan demostrar procedimientos.

## Mensaje de negocio

Clarity no sustituye abogados, peritos ni auditores.  
Clarity les da una base técnica más ordenada, verificable y defendible.
